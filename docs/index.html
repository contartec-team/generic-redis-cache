▸ getJSONGetParams(params)

Returns the list of redis JSON.GET command params

getJSONGetParams(params: (Object | JSONGetParams)): JSONGetParams
Parameters
params ((Object | JSONGetParams) = DEFAULT_JSON_GET_PARAMS) The redis command list/object params
Returns
JSONGetParams: The list of command params
Example
`JSON.GET` params `JSON.GET <key>
   [INDENT indentation-string]
   [NEWLINE line-break-string]
   [SPACE space-string]
   [NOESCAPE]
   [path ...]

paramsObject = {
   options : [ 'NEWLINE', 'SPACE' ]
   paths   : [ 'id', 'nam', 'nested.name' ]
}

paramsArray = [
 'NEWLINE', 'SPACE',
 'id', 'name', 'nested.name'
]

GenericJSONCache.getJSONGetParams(params)
// [ 'NEWLINE', 'SPACE', 'id', 'nam', 'nested.name' ]
